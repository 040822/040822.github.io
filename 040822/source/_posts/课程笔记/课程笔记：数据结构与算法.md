## 1.4 时间/空间复杂度

### 最大子序列问题
法一：暴力搜索，直接求出所有子序列长度并比较。O(n^3)

法二：跑一遍前缀和，然后得出所有子序列长度并比较。O(n^2)

法三：贪心 O(n)  
丢弃掉子序列和为负的子序列。  
```
    maxSum=0
    subSum=0
    for i in range(n):
        subSum+=list[i]
        if(subSum>maxSum):
            maxSum = subSum
        if(subSum<0):
            subSum=0  #在此之前的子序列丢掉
    return maxSum
```

法四：动态规划 O(n)
记以list[i]为右边界的子序列之和最大值为d[i]，则 d[i+1]=max(list[i],d[i]+list[i])  
ps：与贪心一样，子序列小于0就丢弃  
结果为：result = max(d)  
```
    d = [0]*n
    d[0]=list[0]
    maxSum=d[0]
    for i in range(n-1):
        if(d[i]>0):
            d[i+1] = d[i]+list[i+1]
        else:
            d[i+1] = list[i+1]
        if(d[i+1]>maxSum):
            maxSum = d[i+1]
    return maxSum

```

法五：分而治之（挺麻烦的）
