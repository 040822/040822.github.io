强化学习简介

基本概念

重点

智能体(Agent)：我们要训练的AI，能够从外部环境中获取信息，做出动作，并根据反馈来强化自己的策略。
环境（Environment）：智能体所处在的外部系统。
状态(State)：在某一时刻下环境的描述。 状态空间
观察（Observation）：智能体从环境中获取的信息。在完全信息博弈问题中observation=state，非完全信息博弈中不等于。但这两个概念经常混用。
动作(Action)：智能体对环境的一种响应。 动作空间
奖励(Reward)：环境对智能体所做出的某一个动作的反馈信号。
回报（Return）：奖励的加权和。
策略(policy)：智能体中从状态到动作的函数。
强化学习的优化目标：最大化回报Return

强化学习交互的三元组：（s，a，r）
轨迹（tragedy）：交互的一系列经过，s0，a0 r0 s1 a1 r1 。轨迹不一定从状态初始化开始，例如象棋中的残局游戏。

回报：
即时奖励: R(τ)=r0R(\tau)=r_0R(τ)=r0​
累积奖励: R(τ)=∑t=0TrtR(\tau) = \sum\limits_{t=0}^{T} r_tR(τ)=t=0∑T​rt​
折扣回报: R(τ)=∑t=0TγtrtR(\tau) = \sum\limits_{t=0}^{T} \gamma^t r_tR(τ)=t=0∑T​γtrt​

累积奖励=》远视主义
问题：

  • 较早动作的回报可能被高估或者低估
  • 对于无限时间的情况而言，return可能发散
    折扣回报=》取个平衡
    一般gamma的选定取决于你打算让agent看未来的多少步。

价值函数：包括状态价值函数和动作价值函数两种函数，指某一策略在特定情况下能获得 的回报（累积奖励）的估计函数。状态价值函数（V函数）指某一策略在某个特定状态下能获得的回报的估计；动作价值函数（Q函数）指某一策略在某个特定状态下选取某个特定动作时能获得的回报的估计。

  2. TD学习，用1步去学习。
    最优贝尔曼方程：
    Q∗(st,at)=Est+1∼p(.∣st,at)[rt+γmax⁡a∈AQ∗(st+1,a)]Q_*(s_t,a_t)=E_{s_{t+1} \sim p(.|s_t,a_t)}[r_t+\gamma \max\limits_{a\in A}Q_*(s_{t+1},a)]Q∗​(st​,at​)=Est+1​∼p(.∣st​,at​)​[rt​+γa∈Amax​Q∗​(st+1​,a)]

  3. 多步TD：对TD的扩展
    使用m步的数据来拟合Qstar，相比一步的数据，显然可信度更高。
    m步时：
    Q∗(st,at)=E[∑i=1m−1γirt+i+γmmax⁡a∈AQ∗(st+m,a)]Q_*(s_t,a_t)=E[\sum\limits_{i=1}^{m-1}\gamma^i r_{t+i}+\gamma^m \max\limits_{a\in A}Q_*(s_{t+m},a)]Q∗​(st​,at​)=E[i=1∑m−1​γirt+i​+γma∈Amax​Q∗​(st+m​,a)]

DQN的各种变种和改进措施

这里只提一嘴，快速进入PG。

  1. 优先级经验回放：重要的数据样本在训练时抽到的概率更大；使用TD误差的大小来评判数据样本的重要性。
  2. Double DQN：解决Q网络会高估的问题，又增加了一个目标Q网络。
  3. dueling network对决网络：优势函数D∗=Q∗−V∗D_* = Q_* - V_*D∗​=Q∗​−V∗​ ，把Qstar分解为两个网络。
  4. noisy net：给网络参数加噪声
  5. Rainbow：以上各种措施加起来的大杂烩DQN。

基于梯度

REINFORCE

  • 全称：REward Increment = Nonnegative Factor x Offset Reinforcement x Characteristic Eligibility
  • 时间：1987
  • 原论文：Williams R J. Reinforcement-learning connectionist systems[M]. College of Computer Science, Northeastern University, 1987.
  • 后续论文：Williams R J. Simple statistical gradient-following algorithms for connectionist reinforcement learning[J]. Machine learning, 1992, 8: 229-256.


名称由来。
基本思想：使用神经网络来作为策略函数，我们称这个网络为策略网络π(a∣s,θ)\pi(a|s,\theta)π(a∣s,θ)
如何选动作？
把state丢给策略网络，让策略网络输出每个动作的概率分布。离散动作就是每个动作的概率，连续概率的话则将动作概率分布进行建模，并输出概率分布的参数；比如正态分布，输出方差和均值。
然后根据概率分布来进行抽样，获取动作。

如何更新？
重现。

  • 年代：1983
  • 原论文：Barto A G, Sutton R S, Anderson C W. Neuronlike adaptive elements that can solve difficult learning control problems[J]. IEEE transactions on systems, man, and cybernetics, 1983 (5): 834-846.
    基本思想：
    actor：策略网络
    critic：价值网络，这里是Q网络，拟合Q函数。
    使用TD来更新。
