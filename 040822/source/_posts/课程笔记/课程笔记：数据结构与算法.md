## 1.4 时间/空间复杂度

### 最大子序列问题
法一：暴力搜索，直接求出所有子序列长度并比较。O(n^3)

法二：跑一遍前缀和，然后得出所有子序列长度并比较。O(n^2)

法三：贪心 O(n)  
丢弃掉子序列和为负的子序列。  
```
    maxSum=0
    subSum=0
    for i in range(n):
        subSum+=list[i]
        if(subSum>maxSum):
            maxSum = subSum
        if(subSum<0):
            subSum=0  #在此之前的子序列丢掉
    return maxSum
```

法四：动态规划 O(n)
记以list[i]为右边界的子序列之和最大值为d[i]，则 d[i+1]=max(list[i],d[i]+list[i])  
ps：与贪心一样，子序列小于0就丢弃  
结果为：result = max(d)  
```
    d = [0]*n
    d[0]=list[0]
    maxSum=d[0]
    for i in range(n-1):
        if(d[i]>0):
            d[i+1] = d[i]+list[i+1]
        else:
            d[i+1] = list[i+1]
        if(d[i+1]>maxSum):
            maxSum = d[i+1]
    return maxSum

```

法五：分而治之（挺麻烦的）

## 2.2 线性表

定义： n个相同数据类型的元素  
数据之间关系：$<a_i,a_{i+1}>$，（非首尾的）元素有一个前驱元素一个后继元素。  

## 2.3 顺序表

整个顺序表被存储在连续的内存空间里。（当然在物理内存空间里不一定连续）

## 2.4 链表

### 2.4.1 单向链表

head -> ($a_1$|Next)->($a_2$|Next)->...->($a_n$|Next = NULL )


### 2.4.2 双向链表

两个指针，一个next一个prior。

### 2.4.3 循环链表

## 2.6 拓展

广义表：表的元素可以是普通元素，也可以是另一个表。  
多维数组：行优先存放，matlab是列优先。先存完$a[0][0]~a[0][10]$,再存$a[1][0]~a[1][10]$  
特殊矩阵：上三角下三角矩阵只存一半。  
稀疏矩阵：只存非0项，用十字链表。

# 3 栈与队列

## 3.2 栈

后进先出。可以用向量（数组）、单链表实现。这俩入栈出栈时间复杂度都是常数复杂度，但在开辟内存上有区别，顺序表空间固定，链表频繁访问内存，效率低。  
例子：  
元素：node(data,next)，head栈顶指针 #next由栈顶指向栈内部  
方法：pop出栈，push入栈等  

变种：双栈

## 3.3 队列

先进先出，队头删除队尾插入。也可以由顺序表、链表实现。方法还是入队出队。顺序表空间固定，链表频繁访问内存，效率低。  
假溢出：顺序表实现时会出现的问题，队尾指针达到最大值，但由于出队，队头可能还会有空间。  
类型：直的队列和环形队列。  
元素：front队首指针，rear队尾指针。  

变种：  
双端队列：两端都可以插入删除  
超队列：两端插入一端删除  
超栈：两端删除一端插入  

## 3.4 应用

数值转换、表达式解析、递归等。
